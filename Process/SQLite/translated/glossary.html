本页面包含本项目文档中出现但未定义的术语表。这个页面不是教程，只是摘要，很多相关术语未列出。互联网上有无数资源详细介绍此处列出的所有术语。鼓励感兴趣的读者将这些术语输入他们所选的搜索引擎。

按字母顺序排列：
  • DDL：数据定义语言 是与数据库表维护相关的 SQL 子集：CREATE、DROP、ALTER 等。
  • DML：数据操纵语言 是与操作表内容相关的 SQL 子集：INSERT、UPDATE、DELETE 等。
  • DQL：数据查询语言 是与获取结果相关的 SQL 子集，即 SELECT。是否将 INSERT...RETURNING 也归类为 DQL，留待那些正式定义这些术语的人决定。
  • TCL：事务控制语言 是涉及事务的 SQL 子集：BEGIN、COMMIT、ROLLBACK 等。
  • 关系理论 在此上下文中是 SQL 衍生自的研究领域。出于实际原因，SQL 并不严格遵循关系理论，但它是一个接近的近似。
  • SQL：结构化查询语言 是数据库与用户之间的主要接口。它涵盖了 DDL、DML、DQL 和 TCL（见上文）。
