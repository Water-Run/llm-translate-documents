更多信息

SQLite 是免费的，且表现优秀。 大多数人使用 SQLite 时都无需许可证或支持。

SQLite 的 SQLite 论坛 提供免费的支持。这个论坛由包括 SQLite 核心开发团队在内的大型专家社区监控， 他们几乎可以解决你可能遇到的任何 SQLite 问题。

如果你不希望在公开且开放的论坛上讨论问题， 年度维护订阅（“AMS”） 可能更适合你。AMS 让你可以直接与 SQLite 开发者私下通过邮件交流问题， 支持级别与公共论坛相似，但 AMS 包含单方面的保密承诺，因此你可以在没有外界关注的情况下私下讨论问题。 AMS 每年费用为 $1500。

需要更高级支持的用户可以选择 技术支持协议。 技术支持协议会根据每位客户的需求定制， 但通常包括直线电话支持和问题、错误的优先处理。可选提供保证响应时间。 技术支持费用因人而异，通常在每年 $8000 至 $35000 之间。

如果 SQLite 对贵公司而言是“关键任务”， 那么你可能希望成为 SQLite 联盟会员。 SQLite 联盟是由赞助 SQLite 持续开发的公司组成的合作组织， 会员可以获得企业级技术支持、SQLite 开发者的现场访问、无限使用所有授权产品， 以及 SQLite 将继续保持在公有领域、免费且独立的强有力保证，并不会被竞争对手控制。

软件许可证

SQLite 源代码属于 公有领域， 任何人都可以为任何用途免费使用。无需任何许可证。 但有些用户希望获得许可证以获得所有权保证，或仅仅因为公司法务要求。 为此，我们提供 核心 SQLite 源代码的永久许可证与所有权保证。

SQLite 加密扩展 (SEE)、 ZIPVFS 扩展 和压缩与加密只读数据库 (CEROD) 是增强版本的 SQLite，能够处理加密和/或压缩的数据库。SEE 可以读写加密数据库， 并将包括元数据在内的所有数据库内容加密，使数据库文件看起来像白噪声。 ZIPVFS 使用应用提供的压缩与解压函数对数据库进行实时压缩。 CEROD 可以读取经过压缩且可选加密的数据库。 SEE、ZIPVFS 和 CEROD 均以源代码形式提供；许可证持有人需自行为所选平台编译这些产品。 编译这些扩展并不困难。所有产品都以名为“sqlite3.c”的 {汇总源文件} 形式提供。 因此，将 SEE、ZIPVFS 或 CEROD 编译进应用程序， 只需要将启用 SEE、ZIPVFS 或 CEROD 的 sqlite3.c 源文件替换公有领域的 sqlite3.c 源文件并重新编译即可。 SEE、ZIPVFS 和 CEROD 的许可证均为永久性。 这三种扩展都可以读写普通的、未压缩也未加密的数据库文件。

测试服务

测试工具 #3（TH3） 是一套 SQLite 的测试用例，能够在实际部署配置下通过仅使用公开文档接口实现 对核心 SQLite 的 100% 分支测试覆盖率（以及 100% 修改条件/决策覆盖率）。 TH3 被设计用于嵌入式设备，并与 DO-178B 兼容。 每个公共领域的 SQLite 版本都会通过 TH3 进行测试，因此所有用户都会受益于 TH3 测试。 但 TH3 测试本身并非公开。如果硬件或系统制造商希望在其系统上运行 TH3 测试， 可以协商服务协议，由 SQLite 开发者为其运行这些测试。
