dbhash.exe 实用程序

1. 概述

dbhash（在 Windows 上为 dbhash.exe）工具是一个命令行程序，用于计算 SQLite 数据库的架构与内容的 SHA1 哈希值。

Dbhash 忽略多余的格式细节，仅对数据库的架构与内容进行哈希。因此，即使数据库文件经过以下操作修改，哈希仍然保持不变：

  • VACUUM 操作
  • PRAGMA page_size 指令
  • PRAGMA journal_mode 指令
  • REINDEX 操作
  • ANALYZE 操作
  • 通过 备份 API 复制
  • ... 等等

上述操作可能会在原始数据库文件层面造成巨大变化，因此导致文件级的 SHA1 哈希相差甚远。但由于这些操作不会改变数据库文件所表示的内容，dbhash 计算出的哈希也就保持不变。

Dbhash 可用于比较两个数据库，确认它们是等价的，即使它们在磁盘上的表示方式大相径庭。dbhash 还可用于在无需通过较慢链路传输整个远程数据库内容的情况下，验证远程数据库的内容。

2. 使用方法

Dbhash 是一个命令行工具。要运行它，请在命令提示符中键入 “dbhash”，后接一个或多个要进行哈希的 SQLite 数据库文件名。数据库的哈希结果会输出到标准输出。例如： 

drh@bella:~/sqlite/bld$ dbhash ~/Fossils/sqlite.fossil
8d3da9ff87196312aaa33076627ccb7943ef79e3 /home/drh/Fossils/sqlite.fossil

Dbhash 支持命令行选项，可限制要哈希的数据库文件中的表，或仅限对内容或架构进行哈希。运行 “dbhash --help” 可获取更多信息。

3. 构建

要在 Unix 上构建 dbhash 实用程序的副本，请获取一份标准的 SQLite 源代码并执行： 

./configure
make dbhash

在 Windows 上，请执行： 

nmake /f makefile.msc dbhash.exe

dbhash 程序由一个名为 dbhash.c 的 C 代码文件实现。要手动构建 dbhash 程序，只需编译 dbhash.c 源文件并将其与 SQLite 库链接即可。
