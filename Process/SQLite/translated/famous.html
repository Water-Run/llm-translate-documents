SQLite 的知名用户

SQLite 被成千上万的应用程序使用，部署量达数十亿。SQLite 是当今世界上部署最广泛的数据库引擎。

下面按字母顺序列出了一些较为知名的 SQLite 用户。这不是完整列表。SQLite 属于公有领域，因此大多数开发者在自己的项目中使用它时从未告知我们。

Adobe 将 SQLite 用作其 应用程序文件格式， 供 Photoshop Lightroom 产品使用。 SQLite 也是 Adobe Integrated Runtime (AIR) 的标准组成部分。 据报道 Acrobat Reader也使用 SQLite。

Airbus 确认 SQLite 被用于 A350 XWB系列飞机的飞行软件中。

Apple 在运行于 Mac OS-X 桌面与服务器以及 iPhone、iPod 等 iOS 设备上的许多（甚至大多数）原生应用中使用 SQLite。SQLite 也被用于 iTunes，即使是在非 Apple 硬件上。

Bentley Systems 将 SQLite 用作其 应用程序文件格式， 用于 Microstation CAD/CAM 产品。

Bosch 在通用汽车、日产和铃木汽车的多媒体系统中使用 SQLite。（链接）

越来越受欢迎的 Dropbox 文件归档与同步服务据称在客户端主要数据存储中使用 SQLite。

Expensify 在其企业级费用报告软件中将 SQLite 作为服务器端的数据库引擎。

Facebook 在其 osquery 产品中使用 SQLite 作为 SQL 数据库引擎。

我们认为 General Electric 在某些产品中使用 SQLite，因为他们至少四次向 SQLite 开发者询问 SQLite 的美国出口控制编号。所以可以推测 GE 在出口的某些东西中使用 SQLite，但除了 GE 以外似乎没人知道那是什么。

Google 在其 Android 手机操作系统以及 Chrome 浏览器 中使用 SQLite。

据用户错误报告推测，Intuit 在 QuickBooks 和 TurboTax 中使用 SQLite，相关报告见 此处 与 此处。

美国国会图书馆将 SQLite 认定为保存数字内容的推荐存储格式。

McAfee 在其杀毒程序中使用 SQLite。 在此处提及，并在 此处暗示。

Microsoft 将 SQLite 作为 Windows 10 的核心组件以及其他产品的一部分。

SQLite 是 Firefox 浏览器 以及 Thunderbird 邮件客户端 的主要元数据存储格式。

导航数据标准 将 SQLite 用作其应用程序文件格式。

流行的 PHP 编程语言自带 SQLite2 和 SQLite3。

自 Python 2.5 起，所有 Python 发行版都包含 SQLite。

RedHat 包管理器 (RPM) 使用 SQLite 来跟踪其状态。

多次 报道与 记录 显示 Mac OS X 与 Windows 版本的 Skype 客户端中存在 SQLite。

Tcl/Tk 编程语言现在内建 SQLite。SQLite 与 Tcl 配合得特别好，因为 SQLite 最初是一个 Tcl 扩展，之后“逃逸”进入野外。

SQLite 随 Xojo 编程环境（原名 REALbasic）一起捆绑提供。
