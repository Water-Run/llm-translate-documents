SQLite 联盟在 Mozilla 与 Symbian 的创始成员下成立

为世界上部署最广的 SQL 数据库引擎确保独立的技术方向

北卡罗来纳州夏洛特 - 2007 年 12 月 12 日 - 致力于维护 SQLite 作为一个完全开放且独立产品的新会员组织——SQLite 联盟，今日正式宣布成立。Mozilla 与 Symbian Ltd. 已作为创始成员加入 SQLite 联盟。

SQLite 是一个紧凑、高效、高可靠性、可嵌入的 SQL 数据库引擎。SQLite 的源代码处于公共领域，可以免费使用，不收取任何费用。SQLite 是世界上部署最多的 SQL 数据库引擎，目前被广泛应用于领先厂商的各种商业软件产品和电子设备中。现在许多手机、MP3 播放器、机顶盒和个人电脑均内置 SQLite。

SQLite 联盟的使命是持续开发和增强 SQLite，使其成为任何人都可以在不支付版税或许可费的情况下使用的产品。SQLite 联盟成员提供资金以支持该使命，并作为回报获得企业级技术支持。SQLite 的技术控制和方向仍然完全掌握在 SQLite 开发者手中。

Mozilla 是流行的开源 Firefox 浏览器的开发者，Symbian 是领先的开放操作系统，面向高级数据功能智能手机，两者都在其产品中部署了 SQLite 数据库引擎。作为联盟的创始成员，Mozilla 与 Symbian 确保 SQLite 能作为一个可自由访问、属于公共领域的软件资产得到持续开发与支持。

“SQLite 成为受欢迎的嵌入式数据库，是因为它轻量、快速且开源，”Mozilla 工程副总裁 Michael Schroepfer 表示。“作为 SQLite 联盟的创始成员，Mozilla 很高兴能够帮助确保 SQLite 保持充满活力和开放的技术，与我们在互联网上促进选择与创新的使命保持一致。”

“SQLite 联盟将有助于制定对智能手机功能至关重要的数据库管理标准，并将帮助打造一个精通 SQLite 的开发者池，为未来手机开发储备人才，”Symbian 开发者项目与服务总监 Bruce Carney 表示。“我们参与 SQLite 联盟不仅展示了 Symbian 对行业开放标准的承诺，同时，随着手机变得日益强大、智能手机越来越流行，我们关注的是确保从桌面转向移动领域的开发者拥有最轻松、最高效的体验。”

SQLite 曾获得 2005 年 Google/O'Reilly 开源奖。关于 SQLite 联盟的更多信息可访问 SQLite 网站，http://www.sqlite.org/。

关于 SQLite

SQLite 是一个实现了独立、自包含、嵌入式、无服务器、零配置的事务性 SQL 数据库引擎的软件库。SQLite 的代码处于公共领域，可用于任何商业或私人用途。SQLite 当前已被广泛应用于无数软件产品与电子设备中。

SQLite 最初由 D. Richard Hipp 博士于 2000 年开发并发布。该代码在 Hipp 的指导下由一个国际开发团队持续维护和增强。
