SQLite 3.7.14 发行于 2012-09-03

  1. 取消对 OS/2 的内置支持。如果需要将 OS/2 应用程序升级到此版本或更高版本的 SQLite，则需要使用 VFS 并通过 sqlite3_vfs_register() 接口添加应用程序定义的 VFS。本版本中移除的代码 可作为应用程序定义 VFS 的基础。
  2. 确保从命令行 shell的“.dump”命令输出重建数据库时浮点值能够完全保留。
  3. 新增了 sqlite3_close_v2() 接口。
  4. 更新了 命令行 shell，使其可以使用 SQLITE_OMIT_FLOATING_POINT 和 SQLITE_OMIT_AUTOINIT 进行构建。
  5. 改进了 Windows makefile 和构建流程。
  6. 增强了 PRAGMA integrity_check 和 PRAGMA quick_check，使其可以选择只检查某个单独的附加数据库，而不是所有附加数据库。
  7. 增强了 WAL 模式 处理，确保始终至少有一个有效的读取标记，以便只读进程始终可以读取数据库。
  8. 提升了 ORDER BY 和 CREATE INDEX 使用的排序器的性能。
  9. 增加了 SQLITE_DISABLE_FTS4_DEFERRED 编译时选项。
  10. 改进了聚合查询的处理，尤其是聚合函数嵌套在子查询中的情况。
  11. 增强查询规划器，使其在使用 or 优化 的查询上尝试使用 覆盖索引。

  13. sqlite3.c 的 SHA1：5fdf596b29bb426001f28b488ff356ae14d5a5a6
