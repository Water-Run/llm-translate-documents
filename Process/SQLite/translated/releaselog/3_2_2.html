SQLite Release 3.2.2 On 2005-06-12

  1. 添加了 sqlite3_db_handle() API
  2. 添加了 sqlite3_get_autocommit() API
  3. 在解析器中添加了 REGEXP 运算符。标准构建中尚未提供支持该运算符的函数，但用户可以使用 sqlite3_create_function() 添加自定义函数
  4. 提升了速度并缩减了库的占用。
  5. 修复了 64 位架构上的字节对齐问题。
  6. 众多小错误修复和文档更新。

SQLite 发布的 完整列表（单页）以及 事件年表 也可查看。每次提交的详细历史记录可在 SQLite 版本控制站点 查阅。
