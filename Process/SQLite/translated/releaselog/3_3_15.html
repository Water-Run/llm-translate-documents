SQLite 3.3.15 版本（发布于 2007-04-09）

  1. 修复在 3.3.14 中引入的一个错误，该错误导致 CREATE TEMP TABLE 的回滚使数据库连接卡住。
  2. 修复一个错误，该错误在降序查询被数据库更改中断时返回了额外的 NULL 行。
  3. 触发器上的 FOR EACH STATEMENT 子句现在会产生语法错误。以前它会被静默忽略。
  4. 修复一个微妙且相对无害的问题，该问题可能在 I/O 错误后导致资源泄漏。
  5. 测试套件进行了多项改进。测试覆盖率现在超过了 98%。

在一个页面上提供了SQLite 所有发行版的完整列表，并且还提供了时间顺序记录。 每次提交的详细历史可在 SQLite 版本控制站点 查阅。
