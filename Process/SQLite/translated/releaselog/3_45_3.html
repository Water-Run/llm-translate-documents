SQLite 3.45.3 版本 于 2024-04-15 发布

来自 3.45.0 版本（2024-01-15）的先前更改：

  1. 为 SQLITE_RESULT_SUBTYPE 属性添加支持，适用于 应用程序定义的 SQL 函数。所有调用 sqlite3_result_subtype() 的应用程序定义函数必须使用此新属性注册。 否则可能导致 sqlite3_result_subtype() 的调用被视作无效操作。 如果使用 -DSQLITE_STRICT_SUBTYPE=1 编译，当一个未定义 SQLITE_RESULT_SUBTYPE 的函数试图调用 sqlite3_result_subtype() 时将产生 SQL 错误。 对于所有使用子类型的应用程序，建议在编译时开启 -DSQLITE_STRICT_SUBTYPE=1。
  2. 对 JSON SQL 函数 的增强：
    1. 所有 JSON 函数均已重写为使用一种名为 JSONB 的新内部解析树格式。 新的解析树格式可序列化，因此可以存储在数据库中，以避免每次使用 JSON 值时都进行重新解析。
    2. 生成 JSON 的新版本函数现在输出二进制 JSONB，而非 JSON 文本。
    3. json_valid() 函数新增可选第二个参数，用于指定何谓第一个参数“格式正确”。
  3. 向 FTS5 虚拟表添加 FTS5 tokendata 选项。
  4. 默认启用 SQLITE_DIRECT_OVERFLOW_READ 优化。 若要在编译时禁用，可使用 -DSQLITE_DIRECT_OVERFLOW_READ=0。
  5. 查询规划器改进：
    1. 不再允许传递约束优化诱使查询规划器使用范围约束，而当更好的等值约束可用时仍被采纳。 （论坛帖 2568d1f6e6。）
    2. 查询规划器在判定 ANALYZE 识别为低质量的索引时，能更好地忽略这些索引。 （论坛帖 6f0958b03b。）
  6. 将 SQLITE_MAX_PAGE_COUNT 的默认值从 1073741824 提高到 4294967294。
  7. 对 命令行界面（CLI） 的增强：
    1. 提高 Windows 上对 UTF-8 内容的显示效果。
    2. 自动检测 ".dump" 脚本的回放，并相应调整 ".dbconfig defensive off" 及 ".dbconfig dqs_dll on" 等设置。

来自 3.45.1 版本（2024-01-30）的先前更改：

  1. 为了向后兼容，恢复 JSON BLOB 输入漏洞，并承诺在后续版本中继续支持此异常行为。
  2. 修复 PRAGMA integrity_check 命令，使其可在包含 FTS3 和 FTS5 表的只读数据库上运行。 这一问题源于 3.44.0 版本，但直到 3.45.0 发布后才被发现。
  3. 修复与处理损坏的 JSONB 输入相关的问题：
    1. 防止在将损坏的 JSONB 转换为文本时出现指数级运行时间。
    2. 修复在将损坏的 JSONB 转换为文本时可能读取超出 JSONB blob 末尾一个字节的情况。
    3. 使用 jfuzz 增强测试，以防止将来再出现类似 JSONB 问题。
  4. 修复一个长期存在的错误：在使用 内存映射数据库 访问经过巧妙破坏的数据库时， 可能会读取内存映射片段末尾之外的几个字节。
  5. 修复一个长期存在的错误：在 字节码引擎 中可能发生 NULL 指针解引用， 原因是某类专门用于施压查询规划器但实际无意义的 SQL 语句生成了错误字节码。

来自 3.45.2 版本（2024-03-12）的先前更改：

  1. 修复一个在 3.35.0 版本（2021-03-12）的 3a 项增强中引入的 UPSERT 错误，该错误可能导致索引与其表不同步。 论坛贴 919c6579c8。
  2. 缩小在 3.35.0 版本（2021-03-12）的第 8e 项中添加的 NOT NULL 强度降低优化的作用范围。 该优化在某些上下文中无法正常工作，导致查询结果错误。 论坛贴 440f2a2f17。
  3. 其它在前一个补丁版本之后发现的琐碎修正和编译器警告修复。具体详情请参考 时间线。

本次补丁版本 3.45.3（2024-04-15）的更改：

  1. 修复一个长期存在的错误（可追溯至 3.24.0 版本）： 如果某个 UPDATE 触发器 是响应 UPSERT 触发的， 那么它的 "old.*" 值在某些情况下可能不正确。 论坛帖 284955a3cd454a15。
  2. 修复 sum() 函数中的一个错误，该错误可能在应返回无穷大的情况下返回 NULL。 论坛帖 23b8688ef4。
  3. 其它在前一个补丁版本之后发现的琐碎修正和编译器警告修复。具体详情请参考 时间线。
