2006-01-11 发布的 SQLite 3.3.0 版本

  1. CHECK 约束
  2. 在 CREATE/DROP TABLE/INDEX 中使用 IF EXISTS 与 IF NOT EXISTS 子句。
  3. DESC 索引
  4. 更高效的布尔值编码，从而减少数据库文件大小
  5. 更激进的 SQLITE_OMIT_FLOATING_POINT
  6. 将 INTEGER 与 REAL 亲和性分开
  7. 为操作系统接口添加了虚拟函数层
  8. 在 TCL 接口中添加了 “exists” 方法
  9. 改进了对内存不足错误的响应
  10. 数据库缓存可以在同一线程中的连接之间可选地共享
  11. 在数据库连接共享公共缓存时，可选的 READ UNCOMMITTED 隔离（取代默认的 SERIALIZABLE 隔离级别）和表级锁。

还提供了 SQLite 发布完整列表 页面以及一份 年表。 每次提交的详细历史可在 SQLite 版本控制站点 获取。
