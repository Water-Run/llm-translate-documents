SQLite 3.7.0 发行于 2010-07-21

  1. 增加了对 预写日志 的支持。
  2. 查询优化器增强 —— 在能够降低估计查询时间的情况下，会自动创建临时索引。
  3. 查询优化器增强 —— 如果查询同时包含了强制正确输出顺序的 GROUP BY 子句，ORDER BY 将成为无操作。
  4. 增加了 SQLITE_DBSTATUS_CACHE_USED 语义，以配合 sqlite3_db_status()。
  5. 逻辑数据库大小现在存储在数据库头部，以便在不破坏数据库的情况下向文件末尾附加字节，并确保 SQLite 在缺少 ftruncate() 支持的系统上也能正常工作。

在单页上可查阅 完整的 SQLite 发布列表，同时也提供了 年表。 每次提交的详细历史可在 SQLite 版本控制站点 获取。
