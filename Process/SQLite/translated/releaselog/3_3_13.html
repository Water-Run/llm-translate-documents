SQLite Release 3.3.13 On 2007-02-13

  1. 在 sqlite3_analyzer 的输出中添加“碎片化”度量。
  2. 添加 COLLATE 操作符，用于显式设置表达式使用的排序序列。此功能仍在实验阶段，需进一步测试。
  3. 允许连接中最多包含 64 张表——旧限制为 32 张。
  4. 添加了两个新的实验性函数： randomblob() 和 hex()。 它们的预期用途是方便生成 UUID。
  5. 修复了 PRAGMA count_changes 在对带触发器的表执行更新时产生错误结果的问题。
  6. 修复了连接中 ORDER BY 子句优化器的一个 bug，该 bug 会在连接中最左边的表受到 UNIQUE 索引约束时发生。
  7. 修复了 TCL 接口中“复制”方法的一个错误。
  8. fts1 和 fts2 模块的错误修复。

在一个页面中可以查看 完整的 SQLite 发布列表，同样也有一个 年表。 每次提交的详细历史可在 SQLite 版本控制站点 获取。
