SQLite 3.6.5 版本发布于 2008-11-12

  1. 为 journal_mode pragma 添加 MEMORY 选项。
  2. 新增 sqlite3_db_mutex() 接口。
  3. 新增 SQLITE_OMIT_TRUNCATE_OPTIMIZATION 编译期选项。
  4. 修复了 truncate 优化，使 sqlite3_changes() 与 sqlite3_total_changes() 接口以及 count_changes pragma 返回正确的值。
  5. 新增 sqlite3_extended_errcode() 接口。
  6. COMMIT 命令即使在有未完成的查询时也会成功。 当存在未完成的增量 BLOB I/O 请求时，会返回 SQLITE_BUSY。
  7. 当尝试在仍有一个或多个查询待处理时执行 ROLLBACK， 错误码会变更为 SQLITE_BUSY（而非 SQLITE_ERROR）。
  8. 删除对实验性内存分配器 memsys4 与 memsys6 的所有支持。
  9. 新增 SQLITE_ZERO_MALLOC 编译期选项。

提供了一个包含所有 SQLite 发布版本完整列表 的单页视图以及一份 时间线。 每次提交的详细历史记录可在 SQLite 版本控制站点 上查看。
