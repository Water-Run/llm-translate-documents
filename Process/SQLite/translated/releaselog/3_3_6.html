SQLite 3.3.6 版本发布于 2006-06-06

  1. 在 Windows 上与病毒扫描程序配合更好
  2. :memory: 数据库速度更快
  3. 修复 UTF-8 到 UTF-16 转换中一个鲜为人知的段错误
  4. 添加 OS/2 驱动
  5. 为聚合查询返回正确的列元信息
  6. 增强 EXPLAIN QUERY PLAN 的输出
  7. LIMIT 0 现在在子查询上有效
  8. 查询优化器中的错误修复与性能增强
  9. 在 ATTACH 与 DETACH 中正确处理 NULL 文件名
  10. 改进解析器中的语法错误消息
  11. 修复 IN 操作符的类型强制规则

SQLite 的所有版本列表可以在单页面中查看，且还有一个年表。同样，还可以在 SQLite 发布版本完整列表 和 年表 中获取相关内容。每次提交的详细历史记录可在 SQLite 版本控制网站查阅。
