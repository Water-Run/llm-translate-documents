SQLite 3.2.6 发行于 2005-09-17

  1. 修复在数据库超过 1GiB 且 VACUUM（或自动 VACUUM）失败并回滚时可能引起数据库损坏的错误
  2. LIKE 优化现在适用于设置了 COLLATE NOCASE 的列
  3. ORDER BY 和 GROUP BY 现在使用有界内存
  4. 添加对 COUNT(DISTINCT expr) 的支持
  5. 更改 SUM() 处理 NULL 值的方式，以符合 SQL 标准
  6. 在可能的情况下使用 fdatasync() 代替 fsync()，以稍微加快提交速度
  7. 在连接中使用 CROSS 关键字会关闭表重新排序优化
  8. 添加实验性且未记录的 EXPLAIN QUERY PLAN 功能
  9. 在 Windows 上使用 Unicode API

完整的 SQLite 发行版列表 可在单页中查看，并且还有一个 年表。 每次提交的详细历史记录可在 SQLite 版本控制站点 获取。
