SQLite 3.3.4 发布于 2006-02-11

  1. 修复 Unix 互斥实现中的一个失误，该失误可能导致多线程系统死锁。
  2. 修复 64 位机器上的对齐问题
  3. 新增 fullfsync pragma。
  4. 修复优化器漏洞，该漏洞可能导致某些异常的 LEFT OUTER JOIN 返回不正确的结果。
  5. SUM 函数会检测整数溢出，并转换为使用浮点数累积近似结果
  6. 主机参数名称可以以 '@' 开头，以兼容 SQL Server。
  7. 其他杂项错误修复

一个SQLite 发布的完整列表在单个页面中以及一个年表也都可用。 每次提交的详细历史可以在 SQLite 版本控制站点上查看。
