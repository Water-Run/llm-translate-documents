SQLite 3.1.4 版本发布于 2005-03-11

  1. 修复自动清理功能中的错误，如果因违反约束而导致 CREATE UNIQUE INDEX 失败，可能会造成数据库损坏。此问题仅在版本 3.1 引入的新自动清理功能开启时发生。
  2. 如果同步 pragma 设置为非“full”，则禁用 F_FULLSYNC ioctl（目前仅在 Mac OS X 上支持）。
  3. 为未来 3.2 版本的数据库文件格式增加额外的前向兼容性。
  4. 修复形式为 (rowid<'2') 的 WHERE 子句中的错误。
  5. 新增 SQLITE_OMIT_... 编译选项。
  6. 更新 man 手册页。
  7. 从 shell 中移除 strcasecmp() 的使用。
  8. Windows DLL 导出符号 Tclsqlite_Init 和 Sqlite_Init。
