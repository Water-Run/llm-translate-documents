SQLite 3.30.0 发行于 2019-10-04

  1. 为聚合函数添加对 FILTER 子句 的支持。
  2. 为 NULLS FIRST 和 NULLS LAST 语法在 ORDER BY 子句中的使用提供支持。
  3. index_info 和 index_xinfo pragma 已增强，可提供关于 WITHOUT ROWID 表磁盘表示的信息。
  4. 添加了 sqlite3_drop_modules() 接口，允许应用禁用不需要的自动加载虚拟表。
  5. 改进了 CLI 中的 .recover 点命令，使其能从损坏的数据库文件中恢复更多内容。
  6. 增强了 RBU 扩展以支持 表达式索引。
  7. 修改模式解析器，使其在 sqlite_master 表 的 type、name 和 tbl_name 列被破坏且数据库连接未处于 writable_schema 模式时产生错误。
  8. PRAGMA function_list、PRAGMA module_list 和 PRAGMA pragma_list 命令现在在所有构建中默认启用。可通过 -DSQLITE_OMIT_INTROSPECTION_PRAGMAS 禁用。
  9. 为 sqlite3_db_config() 添加了 SQLITE_DBCONFIG_ENABLE_VIEW 选项。
  10. 在 TCL 接口 中添加了 config 方法，以便能够禁用 SQLITE_DBCONFIG_ENABLE_VIEW，并从 TCL 控制其他 sqlite3_db_config() 选项。
  11. 为 应用定义的 SQL 函数 添加了 SQLITE_DIRECTONLY 标志，以阻止这些函数在触发器和视图中使用。
  12. 旧版 SQLITE_ENABLE_STAT3 编译时选项现在是无操作。
