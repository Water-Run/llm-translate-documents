SQLite 发布 3.6.21 于 2009-12-07

  1. 由 sqlite3_trace() 产生的 SQL 输出现在修改为包含 绑定参数 的值。
  2. 针对 SQLite 的某位重要用户的特定用例进行了性能优化。CPU 操作次数减少了 12%（由 Valgrind 测量）。实际性能提升会依工作负载而有所不同。更改包括：
    1. ifnull() 和 coalesce() SQL 函数现在使用内联 VDBE 代码实现，而不是调用外部函数，因此未使用的参数永远无需求值。
    2. substr() SQL 函数在仅计算前缀时不会去测量其整个输入字符串的长度。
    3. 不必要的 OP_IsNull、OP_Affinity 和 OP_MustBeInt VDBE 操作码已被抑制。
    4. 为提升性能进行了各种代码重构。
  3. FTS3 扩展已经进行了重大重构和清理。新的 FTS3 文档 现已提供。
  4. 修复了 SQLITE_SECURE_DELETE 编译时选项，以确保即使在应用 truncate 优化 时内容也会被删除。
  5. 改进了 命令行界面 中“点命令”的处理。
  6. 其他小的 Bug 修复和文档增强。

可以在单页上查看SQLite 发布的完整列表，也可以查看年表。每次检入的详细历史可在 SQLite 版本控制站点 获得。
