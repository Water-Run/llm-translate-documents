SQLite 3.8.10.1 于 2015-05-09 发布

自 3.8.10 版 (2015-05-07) 起的变更：

  1. 添加了 sqldiff.exe 实用程序，用于计算两个 SQLite 数据库文件之间的差异。
  2. 向 FTS3 的 matchinfo() 函数添加了 matchinfo y 标志。
  3. 优化了 ORDER BY、VACUUM、CREATE INDEX、 PRAGMA integrity_check 和 PRAGMA quick_check 的性能。
  4. 修复了在 SQL 模糊测试 中发现的许多晦涩问题。
  5. 在接口文档中标注重要对象的所有方法。 (示例)
  6. 将 American Fuzzy Lop 模糊测试器 作为 SQLite 的 测试策略 的标准组件。
  7. 向 命令行 shell 添加 ".binary" 和 ".limits" 命令。
  8. 在使用 SQLITE_ENABLE_DBSTAT_VTAB 选项编译时，使 dbstat 虚拟表 成为标准构建的一部分。

此补丁版本 3.8.10.1 (2015-05-09) 的变更：

  1. 使 sqlite3_compileoption_used() 响应 SQLITE_ENABLE_DBSTAT_VTAB 编译时选项。
  2. 修复某些 MSVC 版本下 命令行 shell 中的无害警告。
  3. 修复 dbstat 虚拟表 的若干小问题。
  4. SQLITE_SOURCE_ID： "2015-05-09 12:14:55 05b4b1f2a937c06c90db70c09890038f6c98ec40"
  5. sqlite3.c 的 SHA1： 85e4e1c08c7df28ef61bb9759a0d466e0eefbaa2
