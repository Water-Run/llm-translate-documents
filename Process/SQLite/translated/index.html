SQLite 是一个用 C 语言编写的库，提供一个 小巧、 快速、 自包含、 高可靠性、 功能齐全、 SQL 数据库引擎。 SQLite 是全球最常用的数据库引擎。 SQLite 已内置在所有手机和大多数计算机中， 并捆绑在无数人们每天使用的其他应用程序中。 更多信息...

SQLite 源代码 属于公有领域，任何人都可以免费用于任何目的。
