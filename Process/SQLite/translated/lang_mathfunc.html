内置数学 SQL 函数

1. 概述

下面所示的数学函数是标量函数的一个子集，这些函数内置于SQLite amalgamation 源文件中，但只有在使用-DSQLITE_ENABLE_MATH_FUNCTIONS编译时才会激活。

数学函数的参数可以是整数、浮点数，或看起来像整数或实数的字符串或二进制数据。如果任一参数为 NULL，或是无法直接转换为数字的字符串或二进制数据，则该函数将返回 NULL。这些函数在遇到定义域错误时也会返回 NULL，例如尝试对负数取平方根，或计算大于 1.0 或小于 -1.0 的值的反余弦时。

这些函数返回的值通常是近似值。例如，pi() 函数返回的值为 3.141592653589793115997963468544185161590576171875，约比真实值少 1.22465e-16，但这是 IEEE754 双精度所能获得的最接近近似值。

2. 内置标量 SQL 数学函数的说明

acos(X)

返回 X 的反余弦。结果以弧度表示。

acosh(X)

返回 X 的双曲反余弦。

asin(X)

返回 X 的反正弦。结果以弧度表示。

asinh(X)

返回 X 的双曲反正弦。

atan(X)

返回 X 的反正切。结果以弧度表示。

atan2(Y,X)

返回 Y/X 的反正切。结果以弧度表示。根据 X 和 Y 的符号，结果会被放置到正确的象限。

atanh(X)

返回 X 的双曲反正切。

ceil(X)
ceiling(X)

返回大于或等于 X 的第一个可表示整数。对于正数，该例程远离零取整；对于负数，该例程朝向零取整。

cos(X)

返回 X 的余弦。X 以弧度表示。

cosh(X)

返回 X 的双曲余弦。

degrees(X)

将 X 从弧度转换为角度。

exp(X)

计算 e（欧拉常数，约为 2.71828182845905）对 X 的幂。

floor(X)

返回小于或等于 X 的第一个可表示整数。对于正数，该函数朝向零取整；对于负数，该函数远离零取整。

ln(X)

返回 X 的自然对数。

log(X)
log10(X)
log(B,X)

返回 X 的以 10 为底的对数。对于双参数版本，返回 X 的以 B 为底的对数。

兼容性说明：SQLite 的行为与 PostgreSQL 相同，log() 函数计算以 10 为底的对数。大多数其他 SQL 数据库引擎对 log() 计算自然对数。对于双参数版本 log(B,X)，第一个参数是底数，第二个参数是运算对象。这与 PostgreSQL 和 MySQL 的行为一致，但与 SQL Server 相反，后者将第二个参数作为底数，第一个参数作为运算对象。

log2(X)

返回数字 X 的以 2 为底的对数。

mod(X,Y)

返回 X 除以 Y 的余数。这类似于 '%' 运算符，但适用于非整数参数。

pi()

返回 π 的近似值。

pow(X,Y)
power(X,Y)

计算 X 的 Y 次幂。

radians(X)

将 X 从角度转换为弧度。

sin(X)

返回 X 的正弦。X 以弧度表示。

sinh(X)

返回 X 的双曲正弦。

sqrt(X)

返回 X 的平方根。如果 X 为负，则返回 NULL。

tan(X)

返回 X 的正切。X 以弧度表示。

tanh(X)

返回 X 的双曲正切。

trunc(X)

返回位于 X 与 0（含）之间与零距离最远的可表示整数。换句话说，返回 X 的整数部分，向零方向取整。trunc() 函数类似于 ceiling(X) 和 floor(X)，但它始终朝向零取整，而 ceiling(X) 和 floor(X) 则分别向上和向下取整。
