全功能 SQL

不要被名称中的“Lite”所迷惑。SQLite 拥有完整的 SQL 实现，包括：

  • 表、索引、 触发器以及视图 数量不受限制
  • 表中最多可有 32K 列，行数无限
  • 多列索引
  • 索引可以使用 DESC 和 COLLATE
  • 部分索引
  • 表达式索引
  • 聚簇索引
  • 覆盖索引
  • CHECK、UNIQUE、NOT NULL 和 FOREIGN KEY 约束。
  • 使用 BEGIN、COMMIT 和 ROLLBACK 的 ACID 事务
  • 使用 SAVEPOINT、RELEASE 和 ROLLBACK TO 的嵌套事务
  • 子查询，包括 相关子查询
  • 最多 64 路联接
  • LEFT、RIGHT 和 FULL OUTER JOIN
  • DISTINCT、ORDER BY、GROUP BY、HAVING、LIMIT 和 OFFSET
  • UNION、UNION ALL、INTERSECT 和 EXCEPT
  • 丰富的 标准 SQL 函数 库
  • 聚合函数，包括 DISTINCT 聚合
  • 窗口函数
  • UPDATE、DELETE 和 INSERT（这是当然的）
  • 公共表表达式，包括 递归公共表表达式
  • 行值
  • UPSERT
  • 先进的 查询规划器
  • 全文搜索
  • R 树索引
  • JSON 支持
  • IS 运算符
  • 表值函数
  • REPLACE INTO
  • VACUUM
  • REINDEX
  • GLOB 运算符
  • 十六进制整数字面量
  • ON CONFLICT 子句
  • INDEXED BY 子句
  • 虚表
  • 在同一 数据库连接 上使用多个数据库， 通过 ATTACH DATABASE
  • 能够添加 应用定义的 SQL 函数，包括 聚合函数和表值函数。
  • 应用定义的排序函数

还有许多未在上文列出的功能。SQLite 体积小巧，名称中带有“Lite”，但功能绝不逊色。
