High Reliability

SQLite 是一个高可靠性的存储解决方案。它不会出问题。它就是能用。

SQLite 的高可靠性在实践中得到了验证。SQLite 已在全球数十亿部智能手机、物联网设备和桌面应用中无故障使用，几乎已运行了二十年。

SQLite 能优雅地响应系统错误。稀有的内存不足和 I/O 错误会反馈给应用程序。这些错误报告通路都经过细致测试，确保它们始终可用。

SQLite 对损坏的输入具有弹性，包括恶意设计的数据库文件和 SQL 字符串。大量模糊测试确保损坏的输入不会导致崩溃或未定义行为，而是会向应用报告合理的错误。(更多...)

SQLite 构建采用受 DO-178B 启发的流程。SQLite 的测试标准在商业软件中属于最高级别。

SQLite 是开源的，但它不是开放贡献的。SQLite 的所有代码都由一小群专家编写。该项目不接受来自网络匿名路人的“拉取请求”或补丁。

SQLite 的开发者打算持续支持该产品直到 2050 年。为此，源代码被精心记录，以促进长期可维护性。我们更青睐成熟稳定，而非潮流和前沿。

所有这些因素共同造就了一个非常无烦恼的软件库 SQLite。
