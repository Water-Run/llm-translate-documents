关于 SQLite 的图书

SQLite 取证（2018）

作者：Paul Sanderson
出版社：Amazon

这本由著名数字取证专家保罗·桑德森撰写的著作，为调查人员提供了一些分析 SQLite 数据库文件时有用的底层技术细节。

每台电脑和手机都使用数百个 SQLite 数据库，活跃使用中的 SQLite 数据库超过一万亿。因此，在调查中审查这些数据库中的数据（包括尽可能恢复的删除数据）至关重要。这本书完整解释了 SQLite 数据库文件的格式。它展示了记录如何被编码、如何手工解码以及如何解码部分被覆盖的记录。它还描述了 SQLite 的工作原理，尤其是日志和 WAL，如何用于确定仅凭数据本身无法识别的事件。书中涵盖了基本的 SQL 查询以及如何利用它们创建包含来自不同表的数据的自定义报告，并展示了如何使用 SQL 查询来检验关于不同表中数据关系的假设。

本书主要面向取证从业者，假定读者具备一定的计算机取证基础知识；同时也将吸引对 SQLite 文件格式感兴趣的计算机专业人士。

iOS SQLite 学习（2016）

作者：Gene Da Rocha
出版社：Packt Publishing

本书以 SQLite 数据库的架构为起点，并介绍 SQL 的概念。你将具备设计自己数据库系统、管理它并维护它的能力。此外，你还将学会如何使用 SQL 命令平稳操作 SQLite 数据库。

你将能够通过使用其庞大的 C API 调用库扩展 SQLite 的功能，以构建一些有趣、激动人心、全新且智能的数据驱动应用。了解如何使用 Xcode、HTML5 和 Phonegap 构建一个跨平台的现代应用，充分利用这些技术——所有这一切都通过创建一个完整的、可定制的应用骨架来实现，供你用于自己的应用。本书对 SQLite 数据库系统进行了全面描述，详述了设计原则、工程权衡、实现问题以及 SQLite 的运行方式。

SQLite 数据库系统设计与实现（2015）

作者：Sibsankar Haldar
出版社：https://books.google.com/

本书对 SQLite 数据库系统进行了全面描述。它阐述了设计原则、工程权衡、实现问题以及 SQLite 的运行方式。

Android SQLite 精要（2014）

作者：Sunny Kumar Aditya 和 Vikash Kumar Karn
出版社： Packt Publishing

《Android SQLite 精要》聚焦构建数据驱动应用背后的核心概念。本书以同等简明与详尽的方式涵盖基础与高级主题，使读者能够迅速掌握并实现这些概念，从而构建应用数据库。

本书采用实践与示例驱动的方法，帮助读者理解 SQLite 与 Android 数据库驱动应用的核心主题。书中强调以通俗易懂的方式为你提供潜在且广泛适用的开发实践与方法。

移动应用 SQLite 简化指南（2014）

作者：Sribatsa Das
出版社：Amazon
Amazon

《移动应用 SQLite 简化指南》致力于介绍在移动应用中使用 SQLite 数据库的方法与实现流程。它通过逐步示例展示如何创建模式、执行事务并从 Android、BlackBerry 与 iOS 应用访问数据。此外，本书还介绍了如何通过 ADB Shell 与 SQLite 命令行 shell 访问 Android 应用所创建的 SQLite 数据库。对于 BlackBerry 与 iOS 应用，本书则展示了如何使用命令行 shell 访问数据。

SQLite 权威指南（第2版，2010）

作者：Mike Owens 和 Grant Allen
出版社：Apress
Amazon

在企业计算之外，有一种数据库能够让大量软硬件灵活地展现关系型数据库能力，而无需传统数据库管理系统带来的包袱与成本。那就是 SQLite——一款可嵌入的数据库，体积极小，却能处理庞大的数据库。SQLite 配备了通过众多编程与开发环境可用的一系列强大功能，支持 C、Java、Perl、PHP、Python、Ruby、TCL 等语言。

《SQLite 权威指南》第二版 致力于全面覆盖该强大数据库的最新版本，深入介绍了 SQLite 的能力与 API。本书还以 SQLite 为基础，帮助新人迈出数据库开发的第一步。只需短时间，你就能编写各种程序，例如服务器端浏览器插件或下一款优秀的 iPhone 或 Android 应用！

使用 SQLite（2010）

作者：Jay A. Kreibich
出版社：O'Reilly Media
O'Reilly

开发者请注意：数据库已不再只是信息系统组的专属。你可以为桌面、Web、嵌入式系统或操作系统构建数据库驱动应用，而无需依赖 Oracle 与 MySQL 等重量级客户端-服务器数据库。本书展示了如何在开发过程中将 SQLite（一个轻量级的小型数据库）直接嵌入你的应用。现在处理数据的应用拥有巨大优势，而 SQLite 将教你如何开发既能处理数据又保持体积与复杂度可控的数据库驱动应用。本书将引导你完成整个过程，让你快速掌握数据建模、熟悉 SQLite 的 SQL 方言，并学习如何通过脚本语言或基于 C 的语言（如 C# 或 Objective C）使用 SQLite。如今，体量较小但灵活的应用也能参与数据革命。《使用 SQLite》将教会你如何实现。

SQLite 3 - 进入数据库世界（2010）

作者：Kay Droessler
出版社：Lulu.com
Amazon

数据库语言 SQL（结构化查询语言）在数据库中用于定义、操作、保护，主要用于从数据库中查询数据。无论操作系统或复杂、用户友好的图形界面如何，逻辑始终如一。SQLite 是一款免费的桌面数据库，无需费用，适用于多种操作系统，下载与安装快速，并精简到只保留必要部分。对于初学者来说，这是快速入门数据库与数据库语言的最佳条件。练习后若你已产生兴趣，意味着你已经掌握了数据库与 SQL 的大部分知识，因为所讨论内容是可普遍应用于其他众多数据库的基础知识。现在你可以向真正的大型数据库迈进，从专业巨著到庞大的数据库系统。

深入 Symbian SQL（2010）

作者：Ivan Litovski & Richard Maynard
出版社：Wiley
wiley.com

这是关于基于 SQLite 的 Symbian SQL 数据库的权威参考书。作者（均为负责该代码实现的 Symbian 工程团队成员）向你展示如何从内部与外部角度设计代码并简化迁移，还揭示了编写高性能数据库应用的注意事项与禁忌。书中充满资源与示例代码，及时揭示如何设计与调优使用 Symbian SQL 框架的应用，以最终提升性能。

凭借示例代码与内部专业知识，本书拥有让你始终领先一步所需的一切。

SQLite 的 SQL 指南（2009）

作者：Rick F. van der Lans
出版社：Lulu.com
Amazon

SQLite 是一款小巧、快速、可嵌入、基于 SQL 的数据库服务器。它易于安装，无需管理，且为开源。本书详尽介绍了 SQLite。凭借数百个示例以及经过验证的方法与结构，书中教你如何高效且有效地使用 SQLite。它全面描述了 SQLite 3.6 版本所实现的 SQL 方言。本书既可作为教程也可作为参考书。本书中包含的大量 SQL 示例与练习的源代码可从 www.r20.nl 下载。

SQLite 入门（第2版，2009）

作者：Naoki Nishizawa
出版社：Shoeisha
Amazon.jp

本书以流畅的日语专为日本读者撰写。本书为该书的第二版——第一版于 2005 年出版。

深入 SQLite（2007）

作者：Sibsankar Haldar
出版社：O'Reilly Media
O'Reilly

SQLite 是一款小型、零配置、定制化、可嵌入、线程安全、易于维护、面向事务、基于 SQL 的关系数据库管理系统。使用前无需单独的安装或设置过程。无需配置文件。SQLite 为开源且在公共领域可用（欲了解开源，访问 http://opensource.org）。你可以从官网 https://sqlite.org 下载 SQLite 源代码，用你喜欢的 C 编译器编译，然后开始使用编译好的库。SQLite 可在 Linux、Windows、Mac OS X 以及其他一些操作系统上运行，广泛用于中低端数据库应用。本书介绍了设计原则、工程权衡、实现问题以及 SQLite 的运行方式。它全面描述了 SQLite 引擎所有重要组件。

SQLite（2004）

作者：Chris Newman
出版社：Sams
Amazon

SQLite 是一款小巧、快速、可嵌入的数据库。它的流行在于数据库引擎与接口被打包在一个库中，同时还能将所有数据存储在一个文件中。它的功能介于 MySQL 与 PostgreSQL 之间，且比两者都要快速。

在《SQLite》中，作者 Chris Newman 提供了一个全面而实用的指南，涵盖如何使用、管理与编程这款冉冉升起的数据库。如果你想了解 SQLite 或者它与 PHP 的结合使用，这本书就是为你准备的。
