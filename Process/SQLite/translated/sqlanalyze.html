1. The sqlite3_analyzer.exe Utility Program

The sqlite3_analyzer.exe binary 是一个命令行实用工具程序， 用于测量并展示 SQLite 数据库文件中各个表和索引的空间使用量及效率。 示例用法： 

sqlite3_analyzer database.sqlite

输出是人类可读的 ASCII 文本报告，提供关于数据库文件空间利用的 信息。该报告本身旨在便于理解，不过在报告末尾 有对各种被报告参数的 附加解释。

输出同时也是合法的 SQL。报告文本的大部分包含在 一个头部注释中，并且在 报告末尾会有用于创建和初始化数据库的各种 SQL 语句。 构造出的数据库包含生成该报告所用的原始数据。因此可以将原始 报告导入到命令行 shell实例中，然后查询原始数据， 以更深入地了解特定数据库文件的空间利用情况。

1.1. Implementation

The sqlite3_analyzer.exe 程序是一个 TCL 程序，它使用 dbstat 虚拟表来收集数据库文件的信息，并将这些 信息整理成整齐的格式。
