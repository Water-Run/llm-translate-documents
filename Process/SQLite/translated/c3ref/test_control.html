SQLite C 接口

测试接口

int sqlite3_test_control(int op, ...);

sqlite3_test_control() 接口用于读取 SQLite 的内部状态并向 SQLite 注入故障以进行测试用途。第一个参数是操作代码，它决定所有后续参数的数量、含义以及作用。

该接口不供应用程序使用。它仅用于验证 SQLite 库的正确运行。根据 SQLite 库的编译方式，该接口可能不存在。

操作代码的细节、含义、它们接收的参数以及所执行的操作都可能在不通知的情况下更改。与大多数 SQLite API 不同，此函数不能保证在每个版本之间保持一致。

另请参阅 对象、 常量 和 函数。
