SQLite C 接口

结果集中的列数

int sqlite3_column_count(sqlite3_stmt *pStmt);

返回由预处理语句返回的结果集中列的数量。如果此例程返回 0，说明预处理语句不返回数据（例如UPDATE）。然而，仅仅因为此例程返回一个正数，并不意味着一定会返回一行或多行数据。SELECT 语句总会有一个正的 sqlite3_column_count()，但根据 WHERE 子句的限制和表中的内容，它可能不会返回任何行。

另见：sqlite3_data_count()
