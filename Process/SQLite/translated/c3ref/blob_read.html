SQLite C 接口

逐步从BLOB读取数据

int sqlite3_blob_read(sqlite3_blob *, void *Z, int N, int iOffset);

本函数用于将开放的 BLOB 句柄 中的数据读取到调用者提供的缓冲区。N 字节的数据从开放的 BLOB 中、从偏移量 iOffset 开始复制到缓冲区 Z 中。

如果偏移量 iOffset 离 BLOB 末尾少于 N 字节，返回 SQLITE_ERROR，且不读取任何数据。如果 N 或 iOffset 小于零，返回 SQLITE_ERROR，且不读取任何数据。BLOB 的大小（因此 N+iOffset 的最大值）可通过 sqlite3_blob_bytes() 接口确定。

尝试从已过期的 BLOB 句柄 读取数据会失败，并返回 SQLITE_ABORT 错误码。

成功时 sqlite3_blob_read() 返回 SQLITE_OK。否则，返回一个 错误码 或一个 扩展错误码。

此例程仅适用于通过先前成功调用 sqlite3_blob_open() 创建、且尚未被 sqlite3_blob_close() 关闭的 BLOB 句柄。将任何其他指针传入此例程将导致未定义且可能不希望的行为。

另请参见：sqlite3_blob_write()。
