SQLite 的 C 语言接口规范

这些页面旨在提供一个精确而详尽的规范。 想要教程式的介绍，请改为查看：

  • SQLite 五分钟速成 和/或
  • 和 SQLite C/C++ 接口简介。
相同的内容也提供为一个 单一的大型 HTML 文件。

SQLite 接口元素可以分为三类：

  1. 对象列表。 这是 SQLite 库所使用的所有抽象对象和数据类型的列表。总共有数十个对象，但最重要的两个对象是： 一个数据库连接对象 sqlite3 和 一个预编译语句对象 sqlite3_stmt。

  2. 常量列表。 这是 SQLite 使用并在 sqlite3.h 头文件中通过 #define 表示的数值常量列表。这些常量包括各类接口的数值结果代码 （例如 SQLITE_OK）或传入函数以控制行为的标志 （例如 SQLITE_OPEN_READONLY）。

  3. 函数列表。 这是所有在对象上操作并使用或返回 常量的函数和方法的列表。 函数很多，但大多数应用只使用其中一小部分。
