SQLite C 接口

操作系统接口打开文件句柄

typedef struct sqlite3_file sqlite3_file;
struct sqlite3_file {
  const struct sqlite3_io_methods *pMethods;  /* Methods for an open file */
};

一个 sqlite3_file 对象代表在 操作系统接口层 中的一个打开的文件。各个操作系统接口实现可能希望通过附加额外的字段来继承此对象，以满足各自的需求。pMethods 成员是一个指向 sqlite3_io_methods 对象的指针，该对象定义了对打开文件执行 I/O 操作的方法。
