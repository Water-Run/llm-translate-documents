SQLite C 接口

返回数据库连接的文件名

sqlite3_filename sqlite3_db_filename(sqlite3 *db, const char *zDbName);

sqlite3_db_filename(D,N) 接口返回与连接 D 的数据库 N 关联的文件名指针。 如果数据库连接 D 上不存在附加的数据库 N，或者数据库 N 是临时或内存数据库， 则此函数将返回 NULL 指针或空字符串。

此例程返回的字符串值归数据库连接所有并由其管理。 该值将一直有效，直到数据库 N 被 DETACH 或直到数据库连接关闭。

此函数返回的文件名是 VFS 的 xFullPathname 方法的输出。 换言之，即使最初用于打开数据库的文件名是 URI 或相对路径名称， 此处返回的文件名也将是绝对路径名。

如果此例程返回的文件名指针不为 NULL，那么它可作为以下例程的文件名输入参数：

  • sqlite3_uri_parameter()
  • sqlite3_uri_boolean()
  • sqlite3_uri_int64()
  • sqlite3_filename_database()
  • sqlite3_filename_journal()
  • sqlite3_filename_wal()
