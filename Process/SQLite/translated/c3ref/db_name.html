SQLite C 接口

返回数据库连接的模式名称

const char *sqlite3_db_name(sqlite3 *db, int N);

sqlite3_db_name(D,N) 接口返回数据库连接 D 上第 N 个数据库的模式名称的指针，如果 N 越界则返回 NULL。N 值为 0 表示主数据库文件。N 为 1 表示“temp”模式。更大的 N 值对应各种 ATTACH 的数据库。

用于保存 sqlite3_db_name() 返回字符串的空间由 SQLite 自行管理。字符串可能会被任何更改模式的操作释放，包括 ATTACH 或 DETACH，或调用到 sqlite3_serialize() 或 sqlite3_deserialize()，即便这些操作发生在不同线程上。需要长期保留该字符串的应用应自行复制。多个线程同时访问同一数据库连接的应用在调用此 API 时应保护互斥，并在释放互斥锁前将结果复制到自己的私有空间。
