sqlite3_set_authorizer() 接口注册一个回调函数，该函数用于授权某些 SQL 语句操作。回调的第二个参数是一个整数代码，用于指定正在授权的操作。这些是授权回调可能会收到的整数操作代码。

这些操作代码值表示要授权的操作类型。授权回调函数的第三和第四个参数将根据作为第二个参数使用的这些代码中的哪一个而成为参数或 NULL。授权回调的第五个参数是数据库的名称（“main”、“temp”等），如果适用。授权回调的第六个参数是导致访问尝试的最内层触发器或视图的名称；如果该访问尝试直接来自顶层 SQL 代码，则为 NULL。
