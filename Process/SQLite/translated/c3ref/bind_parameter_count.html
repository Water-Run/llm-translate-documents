SQLite C 接口

SQL 参数数量

int sqlite3_bind_parameter_count(sqlite3_stmt*);

此例程可用于查找准备语句中的 SQL 参数数量。SQL 参数是形式为 "?", "?NNN", ":AAA", "$AAA" 或 "@AAA" 的标记，用于作为后续绑定到这些参数的值的占位符。

此例程实际上返回最大的（最右边的）参数索引。对于除 ?NNN 之外的所有形式，这将对应于唯一参数的数量。如果使用了 ?NNN 形式的参数，则列表中可能存在间隙。

另见：sqlite3_bind()、 sqlite3_bind_parameter_name() 与 sqlite3_bind_parameter_index()。
