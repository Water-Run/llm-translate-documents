SQLite C 接口

主机参数名称

const char *sqlite3_bind_parameter_name(sqlite3_stmt*, int);

sqlite3_bind_parameter_name(P,N) 接口返回 预编译语句 P 中第 N 个SQL 参数的名称。 形式为 "?NNN"、":AAA"、"@AAA" 或 "$AAA" 的 SQL 参数的名称分别是字符串 "?NNN"、":AAA"、"@AAA" 或 "$AAA"。 换句话说，最初的 ":"、"$"、"@" 或 "?" 都包含在名称中。 形式为 "?" 且后面没有整数的参数没有名称， 称为“无名”或“匿名参数”。

第一个主机参数的索引是 1，而不是 0。

如果值 N 超出范围或第 N 个参数无名，则返回 NULL。 即使命名参数最初在 sqlite3_prepare16()、 sqlite3_prepare16_v2() 或 sqlite3_prepare16_v3() 中以 UTF-16 指定， 返回的字符串始终为 UTF-8 编码。

另见：sqlite3_bind()、 sqlite3_bind_parameter_count() 以及 sqlite3_bind_parameter_index()。
