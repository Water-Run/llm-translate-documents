CARRAY 表值函数的数据类型

#define SQLITE_CARRAY_INT32     0    /* 数据是32位有符号整数 */
#define SQLITE_CARRAY_INT64     1    /* 数据是64位有符号整数 */
#define SQLITE_CARRAY_DOUBLE    2    /* 数据是双精度浮点数 */
#define SQLITE_CARRAY_TEXT      3    /* 数据是 char* */
#define SQLITE_CARRAY_BLOB      4    /* 数据是 struct iovec */

sqlite3_carray_bind() 接口的第五个参数必须是以下常量之一，用于指定绑定到carray 表值函数中的数组的数据类型。
