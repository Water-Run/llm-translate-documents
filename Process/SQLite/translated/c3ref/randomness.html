SQLite C 接口

伪随机数生成器

void sqlite3_randomness(int N, void *P);

SQLite 包含一个高质量的伪随机数生成器（PRNG），用于在一个已有最大可能 ROWID 的表中插入新记录时选择随机 ROWID。该 PRNG 也用于内置的 random() 和 randomblob() SQL 函数。此接口允许应用程序出于其他目的访问相同的 PRNG。

调用此例程会将 N 字节的随机性存储到缓冲区 P 中。P 参数可以是 NULL 指针。

如果此例程尚未被调用或上一次调用时 N 小于 1 或 P 为 NULL 指针，则使用默认的 sqlite3_vfs 对象的 xRandomness 方法所获得的随机性来为 PRNG 设定种子。如果上一次调用此例程时 N 大于等于 1 且 P 非 NULL，则伪随机性将在内部生成，无需调用 sqlite3_vfs 的 xRandomness 方法。

另请参阅 对象、 常量 和 函数 列表。
