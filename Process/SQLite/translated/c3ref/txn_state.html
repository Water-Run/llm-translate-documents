SQLite C 接口

确定数据库的事务状态

int sqlite3_txn_state(sqlite3*,const char *zSchema);

sqlite3_txn_state(D,S) 接口返回数据库连接 D 中模式 S 的当前 事务状态。如果 S 为 NULL， 则返回数据库连接 D 上任意模式的最高事务状态。事务状态（按从低到高顺序）为：

  1. SQLITE_TXN_NONE
  2. SQLITE_TXN_READ
  3. SQLITE_TXN_WRITE
如果 sqlite3_txn_state(D,S) 的 S 参数不是有效模式的名称，则返回 -1。

另请参阅 对象、 常量 和 函数 的列表。
