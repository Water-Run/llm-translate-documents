SQLite C 接口

结果集中列数

int sqlite3_data_count(sqlite3_stmt *pStmt);

sqlite3_data_count(P) 接口返回预处理语句 P 的结果集当前行中的列数。如果预处理语句 P 没有准备好要返回的结果（通过调用 sqlite3_column() 家族接口），则 sqlite3_data_count(P) 返回 0。sqlite3_data_count(P) 也会在 P 为 NULL 指针时返回 0。如果上一次对 sqlite3_step(P) 的调用返回了 SQLITE_DONE，则 sqlite3_data_count(P) 返回 0。如果上一次对 sqlite3_step(P) 的调用返回了 SQLITE_ROW，则 sqlite3_data_count(P) 会返回非零值，不过在 PRAGMA incremental_vacuum 的情况下它始终返回 0，因为该多步骤 pragma 的每一步都返回 0 列数据。

另请参阅：sqlite3_column_count()
