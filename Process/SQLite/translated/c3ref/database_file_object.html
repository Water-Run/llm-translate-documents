SQLite C 接口

与日志对应的数据库文件

sqlite3_file *sqlite3_database_file_object(const char*);

如果 X 是传递给 sqlite3_vfs 的 xOpen 方法的回滚或 WAL 模式日志文件名称， 那么 sqlite3_database_file_object(X) 会返回一个指向表示主数据库文件的 sqlite3_file 对象的指针。

该例程仅用于自定义 VFS 实现，并非通用接口。 sqlite3_file_object(X) 参数必须是已传递到 sqlite3_vfs.xOpen 方法的文件名指针， 其中 xOpen 的 flags 参数包含 SQLITE_OPEN_MAIN_JOURNAL 或 SQLITE_OPEN_WAL 中的某一位。 该例程的任何其他用法都会导致未定义且可能不良的行为。

另请参阅 对象 列表、 常量 列表 以及 函数 列表。
