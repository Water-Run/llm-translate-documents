SQLite C 接口

预处理语句对象

typedef struct sqlite3_stmt sqlite3_stmt;

该对象的一个实例表示一个已经被编译成二进制形式并准备好执行的单个 SQL 语句。

把每个 SQL 语句看作一个独立的计算机程序。原始 SQL 文本是源代码。预处理语句对象是编译后的目标代码。所有 SQL 必须在运行之前转换成预处理语句。

预处理语句对象的生命周期通常如下：

  1. 使用 sqlite3_prepare_v2() 创建预处理语句对象。
  2. 使用 sqlite3_bind_*() 接口将值绑定到 参数。
  3. 通过调用 sqlite3_step() 一次或多次运行 SQL。
  4. 使用 sqlite3_reset() 重置预处理语句，然后返回第 2 步。此过程可以重复零次或多次。
  5. 使用 sqlite3_finalize() 销毁对象。
