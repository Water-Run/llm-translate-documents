SQLite C 接口

比较两个快照句柄的年龄。

int sqlite3_snapshot_cmp(
  sqlite3_snapshot *p1,
  sqlite3_snapshot *p2
);

sqlite3_snapshot_cmp(P1, P2) 接口用于比较两个有效快照句柄的时间。

如果这两个快照句柄不关联到同一个数据库文件，则比较结果未定义。

此外，只有在从上次删除 wal 文件以来，两个快照句柄都是通过调用 sqlite3_snapshot_get() 获取的情况下，该比较结果才有效。 当数据库恢复到回滚模式或数据库客户端数量降至零时，会删除 wal 文件。 如果任一快照句柄是在 wal 文件最后一次删除之前获取的，则该函数返回的值未定义。

否则，如果 P1 指向的快照比 P2 旧，API 返回负值；如果两个句柄指向相同的数据库快照，则返回零；如果 P1 比 P2 新，则返回正值。

仅当 SQLite 使用 SQLITE_ENABLE_SNAPSHOT 选项编译时，此接口才可用。

另请参阅 对象、 常量 和 函数 列表。
