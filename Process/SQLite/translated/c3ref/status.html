SQLite C 接口

SQLite 运行时状态

int sqlite3_status(int op, int *pCurrent, int *pHighwater, int resetFlag);
int sqlite3_status64(
  int op,
  sqlite3_int64 *pCurrent,
  sqlite3_int64 *pHighwater,
  int resetFlag
);

这些接口用于检索关于 SQLite 性能的运行时状态信息，并可选择重置各种高水位线。第一个参数是要测量的特定参数的整数代码。可识别的整数代码形如 SQLITE_STATUS_...。参数的当前值通过 *pCurrent 返回。记录到的最高值通过 *pHighwater 返回。如果 resetFlag 为真，则在写入 *pHighwater 后最高记录值会被重置。某些参数不会记录最高值。对于这些参数，不会向 *pHighwater 写入任何内容，且会忽略 resetFlag。其他参数只记录高水位线而不记录当前值。对于这些后者参数，不会向 *pCurrent 写入任何内容。

sqlite3_status() 和 sqlite3_status64() 例程在成功时返回 SQLITE_OK，失败时返回非零的 错误代码。

如果当前值或高水位线过大而无法由 32 位整型表示，则 sqlite3_status() 返回的值未定义。

另见：sqlite3_db_status()
