SQLite C 接口

基本数据类型

#define SQLITE_INTEGER  1
#define SQLITE_FLOAT    2
#define SQLITE_BLOB     4
#define SQLITE_NULL     5
#ifdef SQLITE_TEXT
# undef SQLITE_TEXT
#else
# define SQLITE_TEXT     3
#endif
#define SQLITE3_TEXT     3

每个 SQLite 值只有五种基本数据类型之一：

  • 64 位带符号整数
  • 64 位 IEEE 浮点数
  • 字符串
  • BLOB
  • NULL

这些常量是每种类型的编码。

请注意，SQLITE_TEXT 常量在 SQLite 版本 2 中也被使用，但含义完全不同。既链接 SQLite 版本 2 又链接 SQLite 版本 3 的软件应使用 SQLITE3_TEXT，而不是 SQLITE_TEXT。

另请参阅 对象、 常量 和 函数 的列表。
