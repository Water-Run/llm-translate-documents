SQLite C 接口

为虚拟表重载函数

int sqlite3_overload_function(sqlite3*, const char *zFuncName, int nArg);

虚拟表可以使用虚拟表模块的xFindFunction方法提供函数的替代实现。 但这些函数必须存在全局版本才能被重载。

此 API 确保具有特定名称和参数数量的函数存在全局版本。 如果调用此 API 之前尚不存在这样的函数，则会创建一个新函数。新函数的实现总是导致抛出异常。 因此该新函数本身无实际用途。它的唯一目的就是作为一个占位函数，以便被虚拟表重载。

另见 对象、 常量以及 函数的列表。
