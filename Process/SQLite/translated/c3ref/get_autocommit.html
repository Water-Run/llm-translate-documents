SQLite C 接口

自动提交模式测试

int sqlite3_get_autocommit(sqlite3*);

sqlite3_get_autocommit() 接口如果给定数据库连接处于自动提交模式则返回非零，否则返回零。默认启用自动提交模式。自动提交模式由 BEGIN 语句禁用。自动提交模式由 COMMIT 或 ROLLBACK 重新启用。

如果在一个多语句事务中的语句发生某些类型的错误（包括 SQLITE_FULL、SQLITE_IOERR、SQLITE_NOMEM、SQLITE_BUSY 和 SQLITE_INTERRUPT），则事务可能会自动回滚。确定错误发生后 SQLite 是否自动回滚事务的唯一方法是使用此函数。

如果此例程运行期间另一个线程更改了数据库连接的自动提交状态，则返回值未定义。
