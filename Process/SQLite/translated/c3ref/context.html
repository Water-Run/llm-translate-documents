SQLite C 接口

SQL 函数上下文对象

typedef struct sqlite3_context sqlite3_context;

执行 SQL 函数的上下文存储在 sqlite3_context 对象中。指向 sqlite3_context 对象的指针始终是应用定义 SQL 函数的第一个参数。 应用定义 SQL 函数的实现将通过以下调用将该指针传递进去：sqlite3_result()、 sqlite3_aggregate_context()、 sqlite3_user_data()、 sqlite3_context_db_handle()、 sqlite3_get_auxdata()和/或 sqlite3_set_auxdata()。
