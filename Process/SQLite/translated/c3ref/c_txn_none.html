SQLite C 接口

sqlite3_txn_state() 函数允许的返回值

#define SQLITE_TXN_NONE  0
#define SQLITE_TXN_READ  1
#define SQLITE_TXN_WRITE 2

这些常量定义了数据库文件当前的事务状态。 sqlite3_txn_state(D,S) 接口返回这些常量之一，用于描述模式 S 在 数据库连接 D 中的事务状态。

SQLITE_TXN_NONE
SQLITE_TXN_NONE 状态表示当前没有任何事务正在等待。

SQLITE_TXN_READ
SQLITE_TXN_READ 状态表示数据库当前处于只读事务中。内容已经从数据库文件中读取，但数据库文件中的任何内容都尚未改变。如果发生任何更改，且没有其他冲突的并发写事务，事务状态将升级为 SQLITE_TXN_WRITE。执行 ROLLBACK 或 COMMIT 后，事务状态将恢复为 SQLITE_TXN_NONE。

SQLITE_TXN_WRITE
SQLITE_TXN_WRITE 状态表示数据库当前处于写事务中。内容已经写入数据库文件，但尚未提交。下一次 ROLLBACK 或 COMMIT 后，事务状态将变为 SQLITE_TXN_NONE。
