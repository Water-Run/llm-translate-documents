SQLite C 接口

字符串匹配

int sqlite3_strglob(const char *zGlob, const char *zStr);

sqlite3_strglob(P,X) 接口当且仅当字符串 X 匹配 GLOB 模式 P 时返回零。 在 sqlite3_strglob(P,X) 中使用的 GLOB 模式匹配定义与 SQLite 理解的 SQL 方言中 "X GLOB P" 操作符的定义相同。 sqlite3_strglob(P,X) 函数区分大小写。

请注意，该例程在匹配时返回零，不匹配时返回非零，与 sqlite3_stricmp() 和 sqlite3_strnicmp() 的行为一致。

另请参见：sqlite3_strlike()。
