SQLite C 接口

数据库连接状态

int sqlite3_db_status(sqlite3*, int op, int *pCur, int *pHiwtr, int resetFlg);
int sqlite3_db_status64(sqlite3*,int,sqlite3_int64*,sqlite3_int64*,int);

该接口用于检索单个 数据库连接 的运行时状态信息。第一个参数是要查询的数据库连接对象。第二个参数是一个整型常量，从 SQLITE_DBSTATUS 选项 集中选取， 用于确定要查询的参数。SQLITE_DBSTATUS 选项 集可能在 SQLite 的未来版本中继续扩展。

请求参数的当前值会写入 *pCur，最高瞬时值写入 *pHiwtr。如果 resetFlg 为真，则最高瞬时值会重置回当前值。

sqlite3_db_status() 例程在成功时返回 SQLITE_OK，失败时返回非零的 错误码。

sqlite3_db_status64(D,O,C,H,R) 例程的工作方式与 sqlite3_db_status(D,O,C,H,R) 完全相同， 只不过 C 与 H 参数是指向 64 位整数（类型：sqlite3_int64）的指针，而不是指向 32 位整数， 这使得可以返回更大的状态值。如果状态值超过 2,147,483,647，则 sqlite3_db_status() 会截断该值， 而 sqlite3_db_status64() 则会返回完整值。

另见：sqlite3_status() 与 sqlite3_stmt_status()。
