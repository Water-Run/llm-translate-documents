SQLite C 接口

sqlite3_serialize 的标志

#define SQLITE_SERIALIZE_NOCOPY 0x001   /* 不进行内存分配 */

以下一个或多个常量可以对 sqlite3_serialize(D,S,P,F) 的 F 参数进行按位或运算。

SQLITE_SERIALIZE_NOCOPY 表示 sqlite3_serialize() 会返回一个指向其当前正在使用的连续内存数据库的指针，而不会复制该数据库。如果 SQLite 当前没有在使用连续内存数据库，那么此选项会导致 sqlite3_serialize() 返回 NULL 指针。仅当之前通过调用 sqlite3_deserialize() 初始化之后，SQLite 才会使用连续内存数据库。
