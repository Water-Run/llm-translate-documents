SQLite C 接口

销毁预处理语句对象

int sqlite3_finalize(sqlite3_stmt *pStmt);

sqlite3_finalize() 函数用于删除一个 预处理语句。 如果该语句最近一次求值未遇到错误，或者该语句从未被求值过，那么 sqlite3_finalize() 将返回 SQLITE_OK。如果语句 S 最近一次求值失败，那么 sqlite3_finalize(S) 会返回相应的 错误代码或 扩展错误代码。

sqlite3_finalize(S) 例程可以在 预处理语句 S 的生命周期中的任意时刻被调用： 在语句 S 首次求值之前、在一次或多次调用 sqlite3_reset() 之后， 或在任何一次调用 sqlite3_step() 之后，无论该语句是否已经完成执行。

对 NULL 指针调用 sqlite3_finalize() 是一个无害的空操作。

应用程序必须对每个 预处理语句 都进行最终化，以避免资源泄漏。 在语句已经被最终化之后继续使用它是严重错误。在最终化之后使用预处理语句可能会导致未定义和 不良行为，例如段错误和堆损坏。
