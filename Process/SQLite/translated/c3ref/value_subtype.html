SQLite C 接口

查找 SQL 值的子类型

unsigned int sqlite3_value_subtype(sqlite3_value*);

sqlite3_value_subtype(V) 函数返回 应用程序定义的 SQL 函数 参数 V 的子类型。 子类型信息可以用于在一个 SQL 函数和另一个 SQL 函数之间传递少量上下文。使用sqlite3_result_subtype()例程为 SQL 函数的返回值设置子类型。

每个调用此接口的应用程序定义的 SQL 函数在注册函数时， 都应该在文本编码参数中包含SQLITE_SUBTYPE属性。 如果省略了SQLITE_SUBTYPE属性，那么在某些边角情况下，sqlite3_value_subtype()可能会返回零而不是上游子类型。

另见 对象列表、 常量列表，以及 函数列表。
