SQLite C 接口

将数组值绑定到 CARRAY 表值函数

int sqlite3_carray_bind(
  sqlite3_stmt *pStmt,        /* Statement to be bound */
  int i,                      /* Parameter index */
  void *aData,                /* Pointer to array data */
  int nData,                  /* Number of data elements */
  int mFlags,                 /* CARRAY flags */
  void (*xDel)(void*)         /* Destructor for aData */
);

sqlite3_carray_bind(S,I,P,N,F,X) 接口将数组值绑定到 carray() 表值函数 的首个参数之一。S 参数是指向使用 carray() 函数的预处理语句的指针。I 是要绑定的参数索引。P 是 指向要绑定的数组的指针，N 是数组中的元素数量。F 参数是常量 SQLITE_CARRAY_INT32、SQLITE_CARRAY_INT64、 SQLITE_CARRAY_DOUBLE、 SQLITE_CARRAY_TEXT 或 SQLITE_CARRAY_BLOB 中的一个，用于指示要绑定的数组 的数据类型。X 参数如果不是 NULL 指针，则在 SQLite 使用完 P 之后，即便调用 sqlite3_carray_bind() 失败，也会对 P 参数调用 X 函数。特殊的终结器 SQLITE_TRANSIENT 在此无效。

另请参阅 对象、 常量 和 函数 的列表。
