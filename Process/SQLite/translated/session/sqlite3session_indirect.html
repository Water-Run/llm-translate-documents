会话模块 C 接口

设置或清除间接更改标志

int sqlite3session_indirect(sqlite3_session *pSession, int bIndirect);

会话对象记录的每个更改都会标记为直接或间接。如果满足以下任一条件，则该更改被标记为间接：

  • 在更改发生时，会话对象的“间接”标志被设置，或者
  • 该更改不是作为用户 SQL 语句的直接结果发生，而是由 SQL 触发器或外键操作引起。

如果会话中某个行受到多次操作影响，则只有在所有操作都满足上述间接更改的条件时，该更改才被视为间接，否则视为直接。

此函数用于设置、清除或查询会话对象的间接标志。如果传入的第二个参数为零，则清除间接标志。如果大于零，则设置间接标志。传入小于零的值不会修改当前间接标志的值，可用于查询指定会话对象的当前间接标志状态。

返回值指示间接标志的最终状态：若未设置返回 0，若设置返回 1。
