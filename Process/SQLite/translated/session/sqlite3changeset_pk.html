Session 模块 C 接口

获取表主键定义

int sqlite3changeset_pk(
  sqlite3_changeset_iter *pIter,  /* Iterator object */
  unsigned char **pabPK,          /* OUT: Array of boolean - true for PK cols */
  int *pnCol                      /* OUT: Number of entries in output array */
);

对于每个被修改的表，变更集包括以下内容：

  • 表中的列数，和
  • 那些列构成表的主键。

此函数用于查找由迭代器 pIter 当前所指向的变更所修改的表的主键列。如果成功，*pabPK 会指向一个包含 nCol 项的数组，其中 nCol 是该表的列数。如果对应的列属于表的主键，则 *pabPK 的元素被设为 0x01；否则设为 0x00。

如果参数 pnCol 非 NULL，则 *pnCol 会被设置为该表的列数。

如果在迭代器未指向有效条目时调用此函数，则返回 SQLITE_MISUSE，并将输出变量置零。否则返回 SQLITE_OK，并按照上述描述填充输出变量。
