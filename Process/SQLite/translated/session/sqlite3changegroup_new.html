Session 模块 C 接口

创建新的 Changegroup 对象

int sqlite3changegroup_new(sqlite3_changegroup **pp);

sqlite3_changegroup 对象用于将两个或更多 changeset（或 patchset）合并成单个 changeset（或 patchset）。单个 changegroup 对象可以合并 changeset 或 patchset，但不能同时合并两者。输出始终与输入格式相同。

如果成功，该函数返回 SQLITE_OK，并在返回前将 (*pp) 填充为指向新的 sqlite3_changegroup 对象的指针。调用方应在稍后通过调用 sqlite3changegroup_delete() 释放返回的对象。如果发生错误，则返回一个 SQLite 错误码（例如 SQLITE_NOMEM），并将 *pp 置为 NULL。

sqlite3_changegroup 对象的常见使用模式如下：

  • 通过调用 sqlite3changegroup_new() 创建。

  • 通过调用 sqlite3changegroup_add() 向该对象添加零个或多个 changeset（或 patchset）。

  • 应用程序通过调用 sqlite3changegroup_output() 来获取合并所有输入 changeset 的结果。

  • 通过调用 sqlite3changegroup_delete() 删除该对象。

在 new() 和 delete() 之间可以以任意顺序多次调用 add() 和 output()。

除了常规的 sqlite3changegroup_add() 和 sqlite3changegroup_output() 函数外，还可使用流式版本 sqlite3changegroup_add_strm() 和 sqlite3changegroup_output_strm()。
