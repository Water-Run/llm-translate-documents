会话模块 C 接口

删除会话对象

void sqlite3session_delete(sqlite3_session *pSession);

删除之前通过 sqlite3session_create() 分配的会话对象。一旦会话对象被删除， 尝试在任何其他会话模块函数中使用 pSession 的结果是未定义的。

会话对象必须在其附加的数据库句柄关闭之前被删除。有关详细信息， 请查阅 sqlite3session_create() 的文档。

另请参见 对象、 常量 和 函数 的列表。
