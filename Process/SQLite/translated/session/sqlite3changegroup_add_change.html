会话模块 C 接口

向变更组添加单个变更

int sqlite3changegroup_add_change(
  sqlite3_changegroup*,
  sqlite3_changeset_iter*
);

此函数会将作为第二个参数传入的迭代器当前指向的单个变更添加到 changegroup 对象中。添加该变更的规则与 sqlite3changegroup_add() 中的描述相同。

如果变更成功添加到 changegroup，则返回 SQLITE_OK。否则，将返回一个 SQLite 错误代码。

调用此函数时，迭代器必须指向一个有效条目。否则将返回 SQLITE_ERROR，且不会向 changegroup 添加任何变更。此外，迭代器不得使用 SQLITE_CHANGESETAPPLY_INVERT 标志打开。在这种情况下也会返回 SQLITE_ERROR。
