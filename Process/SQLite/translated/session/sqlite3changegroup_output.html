会话模块C接口

从更改组获取组合更改集

int sqlite3changegroup_output(
  sqlite3_changegroup*,
  int *pnData,                    /* OUT: Size of output buffer in bytes */
  void **ppData                   /* OUT: Pointer to output buffer */
);

获取包含更改集（或补丁集）的缓冲区，该更改集表示更改组的当前内容。如果更改组的输入本身是更改集，则输出为更改集。或者，如果输入是补丁集，则输出也为补丁集。

与 sqlite3session_changeset() 和 sqlite3session_patchset() 函数的输出一样，本函数的输出会将与同一表相关的所有更改组合在一起。表的顺序与最初添加到更改组的第一个更改集相同。如果随后加入更改组的第二个或以后的更改集包含在第一个更改集中未出现的表的更改，则这些表会按照首次遇到的顺序附加到输出更改集的末尾。

如果发生错误，则返回 SQLite 错误代码，并将输出变量 (*pnData) 和 (*ppData) 设为 0。否则返回 SQLITE_OK，并分别将输出变量设置为输出缓冲区的大小以及指针。在这种情况下，调用方有责任最终调用 sqlite3_free() 释放缓冲区。

另请参阅 对象、 常量，以及 函数。
