会话模块 C 接口

完成更改集迭代器

int sqlite3changeset_finalize(sqlite3_changeset_iter *pIter);

此函数用于完成通过 sqlite3changeset_start() 分配的迭代器。

此函数应仅在使用 sqlite3changeset_start() 函数创建的迭代器上调用。如果应用程序将通过 sqlite3changeset_apply() 传递给冲突处理程序的迭代器用作参数调用此函数， 则会立即返回 SQLITE_MISUSE 并且调用不会产生任何效果。

如果在调用 sqlite3changeset_xxx() 函数期间遇到错误（例如在 sqlite3changeset_next() 中出现 SQLITE_CORRUPT， 或在 sqlite3changeset_new() 中出现 SQLITE_NOMEM），则此函数会返回与该错误对应的错误代码。否则将返回 SQLITE_OK。这允许使用如下模式（伪代码）：

  sqlite3changeset_start();
  while( SQLITE_ROW==sqlite3changeset_next() ){
    // 对更改执行某些操作。
  }
  rc = sqlite3changeset_finalize();
  if( rc!=SQLITE_OK ){
    // 发生了错误 
  }
